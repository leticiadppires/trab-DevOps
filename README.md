# trab-DevOps

Olá!
A aplicação foi desenvolvida utilizando o framework FastAPI e containerizada com Docker. Foi criado um Dockerfile baseado na imagem python:3.11-slim, com instalação das dependências via requirements.txt. A aplicação foi executada utilizando o servidor Uvicorn e exposta na porta 8000, permitindo acesso à documentação automática via Swagger em /docs.
